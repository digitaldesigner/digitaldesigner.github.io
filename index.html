CSS.

I've built some CSS overrides for the Help Scout app that'll apply some fresh overrides to the left sidebar, and format Properties in the (right) customer sidebar.

Installation

First step is to download Stylebot for Chrome or Firefox. This'll let you inject CSS to any site.

Get some style

Head over to the link below and copy the whole lot to your clipboard:

Help Scout CSS

Applying it

With the code copied, head over to your Help Scout account and open Stylebot and in the "Code" tab (at the bottom), paste all that CSS goodness. You can now close Stlyebox (top X) and you'll be able to see the changes immediately.

That's it!
